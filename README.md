# Bizpark--AI-BE

Backend services:

- `Bizpark.API`: customer-facing API
- `Bizpark.Admin`: template and admin management
- `Bizpark.Runner`: async job processor
- `Bizpark.Commerce`: generic multi-tenant ecommerce service scaffold
- `Bizpark.Core`: shared DTOs/data-layer utilities
